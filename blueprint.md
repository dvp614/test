# 재미난 성격 유형 테스트 (Fun Personality Test) Blueprint

## 프로젝트 개요
이 프로젝트는 사용자가 몇 가지 질문에 답하면 자신의 성격 유형을 알려주는 재미있고 인터랙티브한 웹 애플리케이션입니다. 최신 웹 기술(HTML5, CSS Baseline, Vanilla JS)을 사용하여 프레임워크 없이 빠르고 부드러운 경험을 제공합니다.

## 주요 기능
1. **단계별 퀴즈 진행**: 질문마다 진행 바가 업데이트되며, 부드러운 화면 전환을 제공합니다.
2. **성격 결과 계산**: 사용자의 답변을 점수화하여 여러 유형 중 하나를 도출합니다.
3. **모바일 최적화**: 반응형 디자인을 통해 스마트폰에서도 쾌적하게 이용할 수 있습니다.
4. **모던 UI/UX**: OKLCH 색상 시스템, CSS 변수, `:has()` 선택자 등을 활용한 감각적인 디자인.
5. **결과 공유 (시뮬레이션)**: 결과 화면에서 자신의 유형을 확인할 수 있습니다.

## 디자인 가이드
- **색상**: 생동감 넘치는 OKLCH 기반 파스텔 및 비비드 톤.
- **타이포그래피**: 읽기 편한 시스템 폰트 및 강조된 헤드라인.
- **애니메이션**: 페이드 인/아웃, 버튼 호버 효과, 진행 바 애니메이션.
- **레이아웃**: 중앙 집중식 카드 레이아웃, 여유로운 여백.

## 기술 스택
- **HTML5**: 시맨틱 마크업, `<template>` 태그 활용.
- **CSS3**: Container Queries, Flexbox/Grid, CSS Variables, `:has()`.
- **JavaScript**: ES Modules, Async/Await (필요시), DOM Manipulation.

## 작업 계획
1. **1단계**: `index.html` 구조를 인트로, 퀴즈, 결과 섹션으로 재구성.
2. **2단계**: `style.css`에 퀴즈 전용 스타일 (진행 바, 선택지 버튼, 결과 카드) 추가.
3. **3단계**: `main.js`에 퀴즈 데이터(질문/결과) 및 로직 구현.
4. **4단계**: 모바일 반응형 및 애니메이션 디테일 수정.
5. **5단계**: 최종 에러 체크 및 배포 준비.
