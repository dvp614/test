# SoulAnimal.test - AdSense Professional Edition Blueprint

## 프로젝트 개요
Google AdSense 승인 기준(고유 콘텐츠, 명확한 탐색, 우수한 UX)을 완벽히 충족하는 전문 심리 분석 및 동물 백과사전 플랫폼입니다.

## AdSense 최적화 핵심 구성
1.  **고유 콘텐츠 (The Animal Library)**:
    *   테스트 결과 외에 각 동물(사자, 돌고래 등 8종)에 대한 상세한 성격 분석 및 생태적 특징을 다룬 개별 문서 페이지 추가.
    *   풍부한 텍스트(각 페이지당 최소 500자 이상) 확보로 검색 노출 최적화.
2.  **명확한 탐색 구조 (Navigation)**:
    *   상단 메인 메뉴: 홈, 테스트 시작, 동물 백과사전, 소개.
    *   하단 푸터: 개인정보처리방침, 이용약관, 문의하기, 사이트맵.
3.  **사용자 경험 (UX/UI)**:
    *   **Ad Placeholder**: 광고가 배치될 영역(상단 배너, 본문 사이, 사이드바)을 디자인에 미리 반영.
    *   **Interactive Elements**: 결과 페이지 내 댓글 섹션 및 관련 동물 추천 시스템.
4.  **기술적 SEO**:
    *   JSON-LD 구조화 데이터 적용 (Article, Quiz 스키마).
    *   메타 태그(OG, Twitter Card) 고도화.

## 작업 계획
1.  **1단계**: `blueprint.md` 업데이트 및 폴더 구조 설계.
2.  **2단계**: `index.html` 레이아웃 개편 (메뉴 바, 광고 슬롯, 백과사전 섹션 추가).
3.  **3단계**: `style.css` 고도화 (전문적인 폰트 가독성, 광고 영역 스타일링).
4.  **4단계**: 동물별 상세 페이지(`animal-*.html`) 및 `contact.html` 생성.
5.  **5단계**: `main.js` 로직 수정 (결과 페이지에서 관련 백과사전 링크 연결).
6.  **6단계**: **전체 코드 검토 및 모바일/PC 반응형 최적화** (네비게이션 개선, UI 디테일 강화, 모든 페이지 레이아웃 통일).
7.  **7단계**: 최종 성능 점검 및 `git push`.
