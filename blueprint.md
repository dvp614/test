# 나와 닮은 영혼의 동물 찾기 (Soul Animal Test) - AdSense Ready Blueprint

## 프로젝트 개요
사용자의 성향을 분석하여 8가지 동물 유형 중 하나를 도출하는 고품질 심리 테스트 웹사이트입니다. 구글 애드센스 승인을 목표로 하여 법적 문서, 풍부한 콘텐츠, 모바일 최적화 및 실제 공유 기능을 포함합니다.

## 주요 기능 및 구성
1.  **정밀한 심리 분석**: 12개의 다각도 질문을 통해 성격을 분석합니다.
2.  **8가지 동물 유형**: 사자, 돌고래, 부엉이, 여우, 나무늘보, 골든리트리버, 고양이, 늑대.
3.  **AdSense 최적화**: 
    *   `index.html`: 메인 테스트 페이지.
    *   `privacy.html`: 개인정보처리방침 (AdSense 필수).
    *   `terms.html`: 서비스 이용약관.
    *   `about.html`: 사이트 소개 및 연락처.
4.  **실제 공유 시스템**: `navigator.share()` API를 사용하여 모바일 기기 앱과 연동.
5.  **프리미엄 UI**: OKLCH 컬러 스페이스, 유리창 효과, 부드러운 트랜지션.

## 작업 계획
1.  **1단계**: `blueprint.md` 업데이트 및 사이트 구조 설계.
2.  **2단계**: `index.html`에 네비게이션, 푸터, 법적 링크 추가 및 구조 고도화.
3.  **3단계**: `style.css`에 AdSense 친화적인 전문적인 스타일 적용.
4.  **4단계**: `main.js`에 12개 질문 및 8개 결과 로직, 공유 API 구현.
5.  **5단계**: `privacy.html`, `terms.html`, `about.html` 생성.
6.  **6단계**: 모든 변경 사항 `git push` 완료.
