# CSS Showcase Project Blueprint

## 프로젝트 개요
이 프로젝트는 최신 CSS 기능(Baseline)을 시각적으로 체험하고 테스트할 수 있는 인터랙티브 웹 페이지입니다. 프레임워크 없이 순수 HTML, CSS, JavaScript 및 웹 컴포넌트를 사용하여 구현됩니다.

## 주요 기능 및 디자인
1. **모던 레이아웃**: CSS Grid와 Flexbox를 활용한 유연한 레이아웃.
2. **컨테이너 쿼리 (@container)**: 부모 요소의 크기에 반응하는 컴포넌트 시연.
3. **고급 선택자 (:has())**: 하위 요소 상태에 따라 부모 스타일을 변경하는 지능형 디자인.
4. **현대적 색상 시스템**: `oklch()`를 이용한 생동감 넘치는 컬러 팔레트와 CSS 변수 기반 테이밍.
5. **인터랙티브 컴포넌트**: 웹 컴포넌트(Custom Elements)를 활용한 캡슐화된 UI 요소.
6. **시각 효과**: 레이어드 드롭 쉐도우, 노이즈 텍스처, 세련된 애니메이션.

## 현재 작업 단계
1. `blueprint.md` 생성 및 초기 계획 수립.
2. `index.html` 구조 설계 (섹션별 기능 배치).
3. `style.css` 핵심 스타일 및 변수 정의.
4. `main.js` 인터랙션 및 웹 컴포넌트 등록.
